# Intro
Questo è un micro gestionale per piccole aziende.

## Features
- Creazione contatti
- Organizzazione contatti in gruppi
- Invio newsletter ai gruppi
- Invio documenti/fatture automatico
- Modalità demo
- Opzione HTTPS forzato
